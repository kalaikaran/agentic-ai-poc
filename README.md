# agentic-ai-poc

A TypeScript proof-of-concept for building a simple agentic AI application.

## Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [pnpm](https://pnpm.io/) v10+

## Setup

```bash
pnpm install
```

## Scripts

| Command | Description |
|---|---|
| `pnpm start` | Run the app directly with `ts-node` (no build step needed) |
| `pnpm dev` | Watch `src/index.ts` for changes and restart automatically via `nodemon` |
| `pnpm build` | Compile TypeScript from `src/` → `dist/` |

## Project Structure

```
src/        TypeScript source files
dist/       Compiled output (generated by pnpm build)
```

## Build

Running `pnpm build` compiles all TypeScript source files in `src/` and emits:

- `.js` files → `dist/`
- `.d.ts` declaration files → `dist/`
- `.map` source maps → `dist/`

The entry point after building is `dist/index.js`.

## TypeScript Configuration

Key compiler options (`tsconfig.json`):

- `module: nodenext` — native ESM/CJS interop for Node.js
- `target: esnext` — output modern JavaScript
- `strict: true` — full strict type checking
- `noUncheckedIndexedAccess: true` — safer array/object access
- `exactOptionalPropertyTypes: true` — stricter optional property handling
- `sourceMap / declaration / declarationMap` — enabled for debugging and type consumers
